核心 API 映射（D1 → PostgreSQL）
数据库操作对照表：

Cloudflare D1 (当前)	PostgreSQL (pg 库)
env.DB.prepare(sql).bind(a,b).first()	pool.query(sql, [a,b]) → rows[0]
env.DB.prepare(sql).bind(a,b).all() → {results:[]}	pool.query(sql, [a,b]) → {rows:[]}
env.DB.prepare(sql).bind(...).run() → {meta:{last_row_id}}	pool.query(sql + ' RETURNING id', [...]) → rows[0].id
? 占位符	$1, $2, $3 占位符
示例迁移 — 相册列表：

typescript

// 原 D1 代码 (functions/api/albums/index.ts)
const albums = await env.DB.prepare(`
    SELECT a.*, ... FROM albums a LIMIT ? OFFSET ?
`).bind(pageSize, offset).all();
return jsonResponse(albums.results);
// 新 PostgreSQL 代码 (server/src/routes/albums.ts)
app.get('/api/albums', async (c) => {
    const { rows } = await pool.query(
        `SELECT a.*, ... FROM albums a LIMIT $1 OFFSET $2`,
        [pageSize, offset]
    );
    return c.json(rows);
});
SQL Schema 迁移（SQLite → PostgreSQL）
SQLite 语法	PostgreSQL 语法
INTEGER PRIMARY KEY AUTOINCREMENT	SERIAL PRIMARY KEY
TEXT DEFAULT (datetime('now','localtime'))	TIMESTAMP WITH TIME ZONE DEFAULT NOW()
TEXT (所有字符串)	TEXT / VARCHAR(n)
INTEGER CHECK (x BETWEEN 1 AND 5)	INTEGER CHECK (x BETWEEN 1 AND 5) ✅ 兼容
PRAGMA defer_foreign_keys	删除（PG 原生支持）
CREATE TRIGGER ... AFTER UPDATE	PG 函数 + 触发器语法略不同

存储层替换（R2 → 本地文件系统）
typescript

// server/src/lib/storage.ts
import fs from 'fs/promises';
import path from 'path';
const UPLOAD_DIR = process.env.UPLOAD_DIR || '/app/uploads';
export const storage = {
    async put(key: string, data: Buffer, contentType: string) {
        const filePath = path.join(UPLOAD_DIR, key);
        await fs.mkdir(path.dirname(filePath), { recursive: true });
        await fs.writeFile(filePath, data);
    },
    async delete(key: string) {
        await fs.unlink(path.join(UPLOAD_DIR, key)).catch(() => {});
    },
    // get 由 Nginx 直接处理，不走 Node.js
};
缓存层替换（KV → 内存 Map）
typescript

// server/src/lib/cache.ts
const store = new Map<string, { value: string; expires: number }>();
export const cache = {
    async get(key: string) {
        const entry = store.get(key);
        if (!entry || Date.now() > entry.expires) {
            store.delete(key);
            return null;
        }
        return entry.value;
    },
    async set(key: string, value: string, ttlSeconds: number) {
        store.set(key, { value, expires: Date.now() + ttlSeconds * 1000 });
    },
    async delete(key: string) { store.delete(key); },
};

[NEW] nginx/default.conf
nginx

upstream node_backend {
    server app:3001;
}
server {
    listen 80;
    server_name bbkk.980823.xyz;
    # 前端静态文件
    root /app/dist;
    index index.html;
    # Gzip 压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml;
    gzip_min_length 1024;
    # 图片静态资源（长缓存，Cloudflare CDN 会进一步缓存）
    location /uploads/ {
        alias /app/uploads/;
        expires 1y;
        add_header Cache-Control "public, immutable";
        add_header X-Content-Type-Options nosniff;
    }
    # API 反向代理
    location /api/ {
        proxy_pass http://node_backend;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        client_max_body_size 20M;
    }
    # JS/CSS 静态资源（Vite 带 hash，长缓存）
    location /assets/ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
    # SPA 前端路由 fallback
    location / {
        try_files $uri $uri/ /index.html;
    }
}
